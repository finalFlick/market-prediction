# `.cursor/state/proposed/`

Proposals from the [`reviewer`](../../agents/reviewer.md) agent live here.

Each proposal is a single markdown file named
`<YYYY-MM-DD>-<slug>.md`. Humans ratify by editing the `## Decision`
block, committing the actual change, and appending an entry to
`DECISIONS.md`.

This README exists so the directory is committed even when there are no
pending proposals.
